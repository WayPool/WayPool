4.2 Position Creation Flow

1
User Approves Tokens

User approves USDC and WETH to the WayPool contract

2
Call createMinimalPosition()

User calls the function with tickLower and tickUpper parameters

3
Contract Transfers Tokens

Contract transfers tokens from user and approves to Position Manager

4
Position Manager Mints NFT

Uniswap V3 Position Manager creates the NFT position

5
NFT Sent to User

The NFT position is transferred directly to the user's wallet
